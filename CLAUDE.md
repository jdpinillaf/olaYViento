## Architecture

**Astro 5 + Tailwind CSS v4 static site** for a water sports school in Barranquilla, Colombia. Zero client-side JS by default. Deployed on Vercel.

### Programmatic SEO System

The core architecture is a data-driven page generation system:

1. **`src/data/pages.ts`** — Array of `ProgrammaticPage` objects, each defining a full page (slug, sport, location, SEO meta, content sections, FAQ). Pages have tiers (1=high priority, 2=medium, 3=informational/catch-all).
2. **`src/pages/[slug].astro`** — Single template that renders all programmatic pages via `getStaticPaths()`, pulling from `programmaticPages` array.
3. **`src/data/sports.ts`** — Sport definitions (surf, kitesurf, wing-foil, sup, entrenamiento-funcional) with pricing, includes, descriptions.
4. **`src/data/locations.ts`** — Location definitions (barranquilla, puerto-colombia, colombia-caribe) with coordinates.

To add a new SEO page: add a `ProgrammaticPage` entry to `pages.ts`. The `sportSlug` and `locationSlug` must match existing entries in `sports.ts` and `locations.ts`.

### Layout & Components

- **`src/layouts/BaseLayout.astro`** — Root HTML shell: meta tags, OG/Twitter cards, Google Fonts (Fredoka + Inter), JSON-LD schema injection, WhatsApp floating button.
- **`src/pages/index.astro`** — Homepage assembling section components.
- **`src/components/home/`** — Homepage sections (Hero, Deportes, PorQueNosotros, Temporada, Galeria, Reservar, Ubicacion, FAQ).
- **`src/components/layout/`** — Header (fixed, with floating CTA on scroll) and Footer.

### Schema / Structured Data

`src/lib/schema.ts` builds JSON-LD objects: `Organization`, `WebSite`, `LocalBusiness`, `Course`, `FAQPage`, `BreadcrumbList`. Injected via `<script type="application/ld+json">` in BaseLayout.

### Brand

- Colors defined in `src/styles/global.css` via Tailwind `@theme`: `brand-orange` (#FCA424), `brand-blue` (#54ADDB), `brand-yellow` (#F7DD40), `brand-dark` (#1a1a2e)
- Fonts: Fredoka (display/headings), Inter (body)
